CPR_Game

Overview of CPR_Game
Due note that this game doesn't make the user a certify CPR handler.

The objective for the CPR_Game is to teach the user on how to properly handle a CPR procedure. 



GUI- View


CPR_Model - Model/Controller
	
	The CPR_Model class task is to keep track of the patient's current state during the CPR 	procedure and listen to signals coming from the user input.
